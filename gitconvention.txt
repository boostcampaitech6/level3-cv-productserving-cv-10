Git commit convention 

Fix
잘못된 부분을 수정할 때 사용.

Add
새로운 코드나 문서를 추가할 때 사용.

Remove
코드나 파일을 삭제할 때 사용.

Simplify
코드를 더 간단하게 만들었을 때 사용.

Update
기존에 정상적으로 동작하는 코드를 보완할 때 사용.

Implement
새로운 기능이나 알고리즘 등을 구현했을 때 사용.
큰 단위의 작업에 주로 사용.

Prevent
특정 동작을 방지할 때 사용.

Move
코드나 파일을 다른 위치로 이동시킬 때 사용.

Rename
파일이나 변수 등의 이름을 변경할 때 사용.
